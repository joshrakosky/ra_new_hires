## How It Works

1. User clicks a size button (8oz, 26oz, or 35oz)
2. User selects a color for that size
3. The thumbnail updates to show: `SYKEDT-YETI-{size}-{color}.jpg`
   - Example: 8oz + Black = `SYKEDT-YETI-08-Black.jpg`
   - Example: 26oz + Navy = `SYKEDT-YETI-26-Navy.jpg`
   - Example: 35oz + White = `SYKEDT-YETI-35-White.jpg`
